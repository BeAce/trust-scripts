# trust-scripts

一个简单的命令行版 webpack 配置，0 配置运行 react 项目

## Usage

```sh
// dev mode
$ trust-scripts dev
// prod mode
$ trust-scripts build
```

## Development

```sh
$ cd trust-scripts
$ npm link
$ trust -V
```


// in your test project

```sh
$ npm link trust-scripts
$ trust-scripts dev
// or build
$ trust-scripts build
```
